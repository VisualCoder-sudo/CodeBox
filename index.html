Codebox

Why use Codebox?

Easy to change anything!

Tiny file sizes (Using .json)

View storage literally anywhere!
(as long as you have the .json file)

How can you make files?

1: Name your text

2: Whatever you want to store

3: Click "Upload"

(Optional): You can categorize any file

(Optional): You can set the color of any category
